## 特點

- **可搜尋學生**: 能夠通過學生名字、名字羅馬音，甚至遊戲黑話快速搜尋學生. 🔍️
- **自訂角色**: 發言角色包括老師、學生，也可以上傳圖片自訂角色. 🎅
- **訊息類型豐富**: 除了文字還能發送圖片、貼圖和羈絆劇情框. ❤️
- **靈活的編輯**: 聊天框可隨意插入、刪除、編輯、拖動位置. 👍
- **方便分享**: 對話可以保存為圖片方便分享給小夥伴. 🖼️
- **保留歷史記錄**: 重啟瀏覽器也可以繼續上一次的對話. 📌
- **多語言支持**: 支持中文（簡體和正體）、日語和英語. 🌐
